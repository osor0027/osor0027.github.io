Benefits of Bootstrap

One of the benefits of using bootstrap in that it facilitates communication between designer and developer, since from the design it can be done taking into account the bootstrap layout, which can give us more precision when it comes to development.

Features of Bootstrap

  • the reboot.css file does the job of “reset CSS”
  • It has support for Flexbox.
  • Improvements in the grid system.
  • It has 4 breakpoints
  • The units of measure used are "rem" and "em"
  • Designs adaptable for any device

List Group

If you want to make a list group you should create a div wiht a class of "list-group" and inside of it, you should add the items with a class of "list-group-item"

Badges

I you want to add a badges you should put a span tag with the class of "badge", and if you want to make this one wiht border radius add the class of "badge-pill", and to change the color just type "badge-(color)".

    • Tables

    If you want to create a table you should first create a table tag with the class of "table", then you should create the "thead" and the "tbody" tags and then just start adding the columns and rows. You can change the Background color by putting table-(wiht a color), for example: table-dark.

    Table Border

    If tou want to add a table borders you should type "table-bordered", with that you will see the cells divisions. Also if you want one cell to be longer you should add "colspan="Number of cells that you want that it takes"". There is also a clas to make the rows striped; "table-striped"

    Hover

    If you wanna add hover to the table Just type "table-hover"
